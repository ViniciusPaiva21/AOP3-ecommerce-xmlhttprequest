# AOP3 - E-commerce com XMLHttpRequest

Este projeto tem como objetivo demonstrar o funcionamento do processamento assíncrono de dados utilizando o objeto XMLHttpRequest em uma aplicação web de e-commerce.

## Funcionalidades

- Requisição assíncrona para API REST
- Consumo de dados em formato JSON
- Atualização dinâmica da interface sem recarregar a página
- Listagem de produtos

## 🛠️ Tecnologias utilizadas

- HTML
- JavaScript
- XMLHttpRequest

## Como funciona

1. O usuário clica no botão "Carregar Produtos"
2. O JavaScript cria uma requisição com XMLHttpRequest
3. A requisição é enviada para uma API externa
4. A resposta é recebida em formato JSON
5. Os dados são processados e exibidos na tela
